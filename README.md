# 🎮 Hangman Game

A simple **text-based Hangman Game** developed in **Python** as part of the **CodeAlpha Python Programming Internship**.

## 📌 Project Description

Hangman is a classic word-guessing game where the player tries to guess a randomly selected word one letter at a time. The player has **6 lives**, and each incorrect guess reduces one life. The game ends when the player guesses the entire word or runs out of lives.

This project demonstrates the fundamentals of Python programming, including loops, conditional statements, lists, strings, functions, and user input handling.

---

## ✨ Features

- 🎲 Random word selection
- 🔤 Guess one letter at a time
- ❤️ 6 lives for incorrect guesses
- ✅ Input validation
- 🚫 Prevents duplicate guesses
- 🎯 Win and Lose conditions
- 💻 Console-based interface
